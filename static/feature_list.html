Feature List

Last Updated: 3/22/23

Features to be Implemented

  1. User can choose a language from a drop-down list of a few offered languages
  2. User can use flashcards to learn words in a foreign language along with their English translations
  3. Users can test their knowledge by taking a randomly-generated quiz of 10 questions
  4. User can search for terms in the dictionary tab
  5. User can create an account and log in/out of it
  6. User will see a word of the day and a cycling list of greetings from various languages upon reaching the site

Possible Future Features

  • More learnable languages
  • Audio clips that pronounce the words
  • User can choose from different phrase groups (introductions, numbers, etc.)
